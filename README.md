# Email-Password-Validation-Form
A simple HTML/CSS/JS login form with client-side email and password validation. This project validates user input using regular expressions and displays custom error and success messages without relying on browser pop-ups.

# Features
Email validation using regex (example@domain.com)

Password validation requiring:

Minimum 8 characters

At least one uppercase letter

At least one lowercase letter

At least one number

At least one special character (@$!%*?&)

Custom error messages displayed below inputs

Success message shown when inputs are valid
